## UX: User Experience. O también, "Diseño de Experiencia de Usuario" (UX Design).

- Se trata de brindarle la mejor experiencia posible al usuario al usar nuestra página o aplicación. Es el cómo se SIENTE el usuario al interactuar con la página/app.

- El sentimiento del usuario puede ser positivo o negativo, entonces la idea siempre es que sea lo más positivo posible.

- Para ello, hay que mejorar la calidad de la interacción entre el cliente y el servidor en cada punto de contacto: Simplificar procesos, entregar información relevante, facilitar la navegación, y que el usuario logre fácilmente las tareas que quiera hacer.


## UI: User Interface. O también, "Diseño de interfaz de usuario" (UI Design).

- Es el diseño de la interfaz que va a ver el usuario. Incluye colores, fuentes, imagenes, fondos, botones, íconos...

- Elementos gráficos intuitivos y  visualmente agradables. Se busca una interfaz atractiva, sencilla y coherente.

## UX & UI: La combinación de UX y UI se trata de diseñar interfaces que nos permitan crear la mejor experiencia de usuario.

******************************************************
UX: Cómo se siente usar una página o aplicación.
UI: Cómo se ve esa página o aplicación.
******************************************************

## Extra: La Usabilidad Web

- La Usabilidad Web se refiere a la facilidad del uso de la aplicación. Que el usuario no pueda cumplir con sus objetivos dentro de la aplicación significa que hay un error de usabilidad. Sea cual sea el motivo. 

- Cuanto más alto es el grado de usabilidad, mejor. Eso hará que la experiencia del usuario (UX) sea mejor, y por ende aumentamos la posibilidad de que el mismo vuelva a elegir nuestra aplicación.
