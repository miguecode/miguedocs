> String interpolado

- Cuando hacemos un console.log( ) y escribimos una cadena de texto entre ` `  (alt+96), estamos haciendo un string interpolado. Esas ` `   son comillas invertidas (backticks). 

- Es una interpolación de strings porque cuando escribimos ${ }, adentro de las llaves { } va código JavaScript. Es decir, sería código "embebido". Todo lo que esté dentro de las llaves va a ser código JavaScript. Por eso podemos poner this, nombres de variables, operaciones, y cualquier sintaxis propia de JavaScript.
