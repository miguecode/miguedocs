## Archivo "tsconfig.json"

- El archivo JSON llamado tsconfig.json es un archivo necesario para poder usar TypeScript en nuestro proyecto, en el cual podemos configurar distintas preferencias de uso.

- **Es un objeto con 2 propiedades clave**: "compileOnSave", que es un booleano, y "compilerOptions", que es un objeto lleno de otras propiedades (opciones). Cuando estamos en un proyecto de Angular, lo normal es que también aparezca un "angularCompilerOptions".
