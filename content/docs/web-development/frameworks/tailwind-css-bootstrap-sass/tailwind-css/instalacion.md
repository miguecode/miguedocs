- La instalación de Tailwind es sencilla. Tiene distintas formas de hacerse pero lo ideal es ir a su documentación. Ahí se puede ver muy fácilmente las 4 formas de instalarlo: Con el CLI, la cual es la forma "principal", ya que se instala de forma global, con PostCSS, con un Framework (como Astro, Angular, etc.) o usando CDN (esta es la forma más ligera pero no es recomendada porque sólo sirve para desarrollo y no para producción).

https://tailwindcss.com/docs/installation
