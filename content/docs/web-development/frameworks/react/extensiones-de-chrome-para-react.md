Chrome tiene milliones de extensiones disponibles, pero estas son las 3 extensiones que no me pueden faltar a la hora de usar React.

1️⃣ React Scan

Detecta automáticamente problemas de rendimiento en tu aplicación de React.

No requiere cambios en el código, resalta exactamente los componentes que necesitas optimizar y está disponible a través de una etiqueta script, npm y muchas más.

2️⃣ React Developer Tools

Te permite inspeccionar las jerarquías de componentes de React en las herramientas para desarrolladores de Chrome con dos nuevas pestañas: Components y Profiler.

La pestaña Components muestra los componentes raíz de React que se renderizaron en la página, así como los subcomponentes que generaron.

La pestaña Profiler te permite registrar información sobre el rendimiento.
